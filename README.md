# 4C2025_Blender-Animaton
2025计算机设计大赛_数媒动漫与短片与短片_动画普通组_千年造化应星时
动画标题：千年造化应星时
主题：
本动画以中国古代物理成就杰出科学家宋应星为主角，通过一个架空性的故事，以《天工开物》内容为主要参考，部分参考了其他我国古代物理成就典籍，简要地介绍了一些我国古代机械和工艺方面的物理成就，内容力求兼具趣味性和科普性，旨在弘扬中华优秀科学家精神和中华优秀自然科学文明和文化传承。
内容简介：
崇祯八年，宋应星赴袁州府分宜县任教。偶然间听闻当地有精怪作乱，影响到了当地生产，宋应星亲自前往调查，在追踪精怪到一处破旧道观时发现一奇异的竹简。竹简自称为黄帝所作，专司记录人民创造之器具，于千年来行走人间，但在途径道观时一时不察却被竹简中记录的部分器具所化之灵逃出。故委托宋应星助其收回逃散的器具之灵。而回收器具的办法就是破坏其功能或者载体。宋应星考虑到器具毕竟为民生所用之物，故决定探究丢失器具的原理来逆向推导破坏其功能的办法，以此减少损失。
于是宋应星查阅研究和求访各种丢失器具的原理，并在这个过程中见识到我国古代劳动人民的伟大智慧和物理成就，决定将这些工艺知识和器具记录下来编辑成书，普及开来流传后世，故数年之后，《天工开物》（1637）出世。
剧情线：宋应星听闻精怪作乱，追踪调查→在道观遇到竹简所化老人，制服织机上的造物之灵→接受竹简委托，收服其他的造物之灵（水车、陶瓷、高温烧制工艺）→最终宋应星有感而发，决定着手撰写《天工开物》。
涉及到的机器与工艺：花缕织机、水车、陶瓷与高温烧制工艺
